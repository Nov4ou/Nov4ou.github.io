Proof of $ \lim\limits_{x \to 0} \frac{sin(x)}{x} = 1 $

The proof has to rely on the geometry of right-angled triangles, since that's where the sine function comes from. Let's start with the right-hand limit (as $x \to 0^+$).
Assume that $x$ is near $0$ but positive, draw a wedge $OAB$ of a circle of radius $1$ unit with angle $x$:



The area of the whole circle is $\pi r^2=\pi$ square units and the area of the wedge is $(\frac{x}{2\pi} \times \pi = \frac{2}{x}$ square units.
First draw in the line $AB$. Then drop a perpendicular from $A$ down to the line $OB$; call the base point $C$. Also extend the line $OA$ out a little bit, and finally draw the tangent line to the circle at the point $B$. That tangent line intersects the extended line $OA$ at a point $D$. Properties of Asymptotic Notations

Posted on 2022-08-13 Edited on 2022-08-30 In Introduction to algorithm, 3th

Asymptotic Notations:

\begin{align}
\Theta(g(n))=\left.\{ f(n): there \ exist \ positive \ constants \ c_1, \ c_2, \ and \ n_0 \right. \\\\
\left.\ such \ that \ 0 \leq c_1g(n) \leq f(n) \leq c_2g(n) \ for \ all \ n \geq n_0 \right\}.
\tag{1}
\end{align}
\begin{align}
O(g(n))=\left.\{ f(n): there \ exist \ positive \ constants \ c \ and \ n_0
\right. \\\\
\left.\ such \ that \ 0 \leq f(n) \leq cg(n) \ for \ all \ n \geq n_0 \right\}.
\tag{2}
\end{align}
\begin{align}
\Omega(g(n))=\left.\{ f(n): there \ exist \ positive \ constants \ c \ and \ n_0
\right. \\\\
\left.\ such \ that \ 0 \leq cg(n) \leq f(n) \ for \ all \ n \geq n_0 \right\}.
\tag{3}
\end{align}
\begin{align}
o(g(n))=\left.\{ f(n): for \ any \ positive \ constant \ c>0, \ there \ exist
\right. \\\\
\left.\ a \ constant \ n_0>0 \ such \ that \ 0 \leq f(n) < cg(n) \ for \ all \ n \geq n_0 \right\}.
\tag{4}
\end{align}
\begin{align}
\omega(g(n))=\left.\{ f(n): for \ any \ positive \ constant \ c>0, \ there \ exist
\right. \\\\
\left.\ a \ constant \ n_0>0 \ such \ that \ 0 \leq cg(n) < f(n) \ for \ all \ n \geq n_0 \right\}.
\tag{5}
\end{align}
